Today I got opportunity to re-think my dev principals. So I listed them here.

# My Dev Principals
- Simplicity
- Readability
- Visibility
- Less code with utilizing  libraries
- Well tested
- Following YAGNI
- Following DRY
- Frequent deliveries
- Prioritized testing
- Refactoring on working on a new issue
